# Stakeholders

## definition

дословно акционеры, лица, которые инициируют проект (бизнес-заказчики), которым скрам-проект будет приносить выгоду. 

Они вовлечены в [[Scrum|скрам]] только во время обзорного совещания по спринту ([[Sprint Review]])
